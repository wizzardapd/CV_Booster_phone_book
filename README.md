# CV_Booster_phone_book

A small contact management application built with C# and .NET as a practice project to improve my understanding of object-oriented programming, CRUD operations, and basic software architecture.

The application runs in the console and allows users to manage contacts through a simple menu interface.

# Features

- Add new contacts
- View all contacts
- Update existing contacts
- Delete contacts
- Search contacts
- Export contacts to CSV

# What I Practiced

This project was mainly focused on learning:

- CRUD operations
- Class design
- Constructors
- Separation of concerns
- Working with collections
- Basic software architecture
- User input handling
- CSV exporting

I also tried to structure the project in a cleaner way by separating:
- UI logic
- Business logic
- Data models
